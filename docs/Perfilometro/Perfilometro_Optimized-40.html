Changing the view

You have a choice of 5 different views for display of the measured values in the visualisation
area (in 

Parametrization

 mode). The views offer you the information required for parametrisa-

tion of the respective situation. To change the view in the visualisation area, proceed as follows:

Instruction:

w

Select the desired view in the drop-down menu on the top left of the visualisation area.

The bottom diagram shows the profile points of the object. The field of view comes with a grey

measurement field is no longer available for further processing.

Optionally, the displayed profile can be saved as a CSV file via the web interface. The CSV file
