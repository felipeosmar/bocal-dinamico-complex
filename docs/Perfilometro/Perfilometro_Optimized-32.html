Set up RS485 interface with Modbus RTU

The sensor supports Modbus RTU and ASCII via RS485 for retrieving measured values and for
parameterization.

Communication via the RS485 interface is a serial master-slave communication, reason why the
serial communication parameters must first be known for all users:
