Mode Save Parameter-Setups

You can save up to 32 parameter setups in the 

Save Parameter-Setups

 mode. In addition, you

have the option to export and re-import parameter setups. External storage is in the .json for-
mat. The JSON file can be transferred to sensors with identical type code (measurement range
and interface). The parameter setups can be renamed individually using the Web interface. Indi-
vidual parameter setups can be saved, load or deleted. They are referenced via the process in-
terfaces through use of a unique number, which can also be displayed in the Web interface.
