## Project Overview

This is a hardware control project ("bocal dinamico" - dynamic nozzle) involving:
- **Raspberry Pi 4** as the control platform
- **mightyZAP actuators** controlled via MODBUS protocol (see FC_MODBUS manual)
- **Profilometer** measurement equipment integration

## Reference Documentation

- `FC_MODBUS_mightyZAP-User-Manual_ENG_23H08_V3.4.pdf` - MODBUS protocol reference for mightyZAP linear actuators
- `Perfilometro.pdf` - Profilometer equipment documentation
- `raspberry pi 4 pinout.png` - GPIO pinout reference for hardware connections
