Save Parameter-Set-
ups

n

Storage of the the set parameters as a parameter setup in the
sensor (maximum 32 setups) or externally.

n

External storage is in the .json format. The JSON file can be
transferred to sensors with identical type code (measurement
range and interface).

n

The parameter setups can each be renamed separately as
well as imported/exported.

n

Alternatively, you can import/export the entire parameter
setup.

n

A single file contains all sensor parameters except the inter-
face-relevant parameter.

n

Interface-relevant parameters are also saved when saving the
entire parameter setup block. When importing, you can select
whether these are imported.

Device Configuration

n

Setting the interface-specific properties (Ethernet configura-
tion, active process interfaces).

Condition Monitoring

n

Display of diagnostic data, such as operating time, tempera-
ture and operating voltage.
