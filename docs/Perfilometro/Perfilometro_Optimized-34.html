8

Web interface

Web interface access is directly via web browser, so sensor configuration and operation does
not require any additional software.

The web interface is a simple and yet precise way for sensor configuration. The web interface
can be used for the configuration of sensor parameters as well as parametrisation of the appli-
cation-specific measuring tasks. The web interface allows you to visualise what the sensor
"sees", so you can use this information to precisely adjust the sensor to the given conditions.

For a detailed description of the web interface, the individual elements of the user interface and
all required operating processes, see Web interface.
