6.3

Connecting the sensor to electricity

NOTICE

Use a power unit for sensor supply. The USB interface is intended for data transfer only. Power
supply is always via the 12-pin M12 connector.

Instruction:

a) Ensure that the system is disconnected from power.

b) Connect the sensor according to the pin assignment.
